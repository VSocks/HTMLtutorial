Intrudoção básica a repositórios e controle de versões

O que é git?

Git é um sistema de controle de versões, ou seja, um software utilizado para desenvolver programas que oferece um aplo controle sobre cada mudança a ser feita no código. Desse modo, é possível que diversas pessoas trabalhem sobre o mesmo projeto e ao mesmo tempo e ultimamente juntem seus trabalhos em um só, e que cada erro que seja cometido durante a programação possa ser revertido para uma versão anterior, entre outras aplicações.

O que é github?

Github é o repositório do git, ou seja, é uma plataforma em nuvem em que cada pessoa possui seu perfil onde podem armazenar seus projetos e colaborar com outros desenvolvedores.
